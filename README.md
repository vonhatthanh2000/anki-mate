# BoostVocab

BoostVocab is a playful and creative vocabulary-building app designed to help users enhance their language skills in an engaging way. With a hand-drawn, sketch-like interface, BoostVocab makes learning new words fun and interactive.

## Features

- **User-Friendly Interface**: Navigate easily through a visually appealing, informal design that feels approachable.
- **Add Words and Meanings**: Effortlessly add new vocabulary words along with their meanings.
- **Highlighting Tool**: Write paragraphs using your vocabulary words, with instant highlighting for easy reference.
- **Playful Aesthetic**: Enjoy a unique design that fosters a relaxed learning environment.
